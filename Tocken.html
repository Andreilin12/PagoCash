Guía de Uso

¿Cómo funciona el sistema de tokens?

Este sistema permite compartir datos entre usuarios de forma segura mediante tokens temporales:

  • Token de Acceso: Es un código único que generas para dar acceso a tus datos.
  • Validar Token: Al ingresar el token de otro usuario, solicitas acceso a sus datos.
  • Seguridad: Los tokens expiran después de 1 minuto y se renuevan automáticamente.

Proceso de vinculación

  1. Paso 1: El dueño de los datos (Usuario A) copia su token.
  2. Paso 2: El usuario que quiere acceso (Usuario B) ingresa ese token.
  3. Paso 3: El Usuario B obtiene acceso a los datos del Usuario A.
  4. Paso 4: El Usuario A puede revocar el acceso en cualquier momento.

Recomendaciones de seguridad

  • Nunca compartas tu token con personas no autorizadas.
  • Solo ingresa tokens de usuarios en los que confíes.
  • Revoca el acceso inmediatamente si sospechas de actividad no autorizada.
  • Los tokens expiran automáticamente después de 1 minuto si no se usan.

Administrar accesos

En la sección "Accesos Concedidos" puedes:

  • Ver quién tiene acceso a tus datos.
  • Revocar accesos cuando ya no sean necesarios.
  • Identificar cuándo fue la última actividad del usuario.

¿Qué sucede al revocar acceso?

Cuando revocas el acceso a un usuario:

  • El usuario pierde inmediatamente el acceso a tus datos.
  • Tus datos desaparecen de su cuenta.
  • La relación de vinculación se elimina por completo.
  • Puedes volver a conceder acceso generando un nuevo token si es necesario.
