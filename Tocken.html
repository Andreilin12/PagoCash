Guía de Usuario

Funcionamiento de los Tokens

Tokens Temporales

Tu token de vinculación se renueva automáticamente cada 60 segundos para mayor seguridad.

Compartir Token

Usa el botón "Copiar Token" para compartir tu código con otro usuario que quieras vincular.

Vincular Cuentas

Para vincular con otro usuario, ingresa su token en el campo correspondiente y haz clic en "Validar Token".

Usuarios Vinculados

Cuenta Principal

El primer usuario en generar el token se convierte en la cuenta principal (indicado con el badge "Principal").

Compartir Datos

Al vincular cuentas, ambos usuarios tendrán acceso a los mismos datos y préstamos.

Desvincular

Puedes desvincular a un usuario en cualquier momento usando el icono . Esto eliminará todo acceso compartido.

Información Adicional

Seguridad

Los tokens son temporales y únicos, lo que garantiza que solo las personas con tu token actual puedan vincularse.

Re-vincular

Si desvinculas a un usuario, puedes volver a vincularlo en cualquier momento usando un nuevo token.
