Guía de Uso

¿Cómo funciona el sistema de tokens?

Este sistema te permite compartir tus datos con otros usuarios de forma segura mediante tokens temporales.

1. Compartir tus datos

Para permitir que otro usuario acceda a tus datos:

  • Copia tu token de vinculación (válido por 60 segundos)
  • Comparte este token con el usuario que quieres que acceda a tus datos
  • El token se renovará automáticamente cada minuto por seguridad

2. Acceder a datos de otro usuario

Para acceder a los datos de otro usuario:

  • Solicita al dueño de los datos que te comparta su token
  • Ingresa el token en el campo "Solicitar acceso a otro usuario"
  • Haz clic en "Validar Token y Solicitar Acceso"
  • Si el token es válido, tendrás acceso a los datos de ese usuario

3. Administrar accesos

En la sección "Accesos Concedidos" puedes:

  • Ver quién tiene acceso a tus datos
  • Revocar el acceso cuando lo desees
  • Ver el estado de actividad de cada usuario

Seguridad

Por tu seguridad:

  • Los tokens expiran después de 60 segundos
  • Solo comparte tu token con usuarios de confianza
  • Puedes revocar accesos en cualquier momento
  • Al revocar acceso, el usuario perderá inmediatamente el acceso a tus datos

Notas importantes

  • El dueño de los datos siempre mantiene el control
  • Al acceder a datos de otro usuario, verás sus datos pero los tuyos permanecerán privados
  • Puedes tener acceso a múltiples usuarios simultáneamente
